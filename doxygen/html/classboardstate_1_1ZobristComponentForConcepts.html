Detailed Description

template<typename KeyType, size_t NumConfKeys>
class boardstate::ZobristComponentForConcepts< KeyType, NumConfKeys >

Container for one or more boardstate::ZobristCalculatorForConcepts objects.

Using more than one boardstate::ZobristCalculatorForConcepts (i.e. NumConfKeys > 0) allows hash collisions to be detected.
