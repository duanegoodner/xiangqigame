Detailed Description

template<typename KeyType, size_t NumConfKeys>
class boardstate::ZobristSummarizer< KeyType, NumConfKeys >

Implements the BoardStateSummarizer interface, providing a moveselection::MinimaxMoveEvaluator with "reasonably unique" hash values for each encountered board state, and the ability to read/write board_state-MinimaxResult pairs in a boardstate::TranspositionTable.
